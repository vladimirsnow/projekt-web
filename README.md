
# GameZone

Проект учебного сайта **GameZone** — выполнен полностью на HTML и CSS (без JavaScript).

## Состав проекта

- **index.html** — главная страница с CSS-модальным окном, CSS-слайдером и переключателем темы (тёмная/светлая).
- **reviews.html** — каталог с карточками обзоров.
- **about.html** — информация о сайте.
- **contacts.html** — форма обратной связи.
- **top.html** — страница топа игр.
- **css/**
  - `reset.css` — сброс стилей
  - `style.css` — основной дизайн
  - `mobile.css` — адаптивность
- **img/** — заглушки изображений (можно заменить своими webp/png).

## Возможности

- Переключатель темы (`checkbox` → CSS variables)
- CSS-слайдер без JS (`radio` + `transform`)
- CSS-модальное окно (`checkbox` hack)
- Полностью адаптивный дизайн (480/768/1024)
- Использованы Flexbox и Grid

## Использование

1. Распакуйте архив.
2. Откройте `index.html` в браузере.
3. При необходимости замените изображения в папке `img/`.
